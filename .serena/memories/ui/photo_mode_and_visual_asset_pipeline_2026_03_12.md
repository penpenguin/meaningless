2026-03-12: Added a premium visual asset pipeline and photo mode. New `src/assets/visualAssets.ts` defines `AssetManifest`/`VisualAssetBundle`, preloads static SVG textures from `public/assets/aquarium/`, and marks them as shared assets. `AdvancedAquariumApp` now preloads visual assets before scene construction and passes them into `AdvancedAquariumScene`. `AdvancedAquariumScene` accepts optional visual assets, forwards them to `AquascapingSystem` and `DetailedFishSystem`, and adds `setPhotoMode()` which slows motion (`motionScale=0.72`), enables auto-rotate, and lerps camera framing toward a showcase angle. Game state now persists `photoModeEnabled` in `GameSave.profile.preferences`; reducer action `SETTINGS/SET_PHOTO_MODE` toggles it and hides/restores the HUD. `GameHudOverlay` adds a Photo Mode toggle and repurposes the reveal tab to exit photo mode when active. `AquascapingSystem` now uses external textures for leaf, driftwood, and rock materials when available. `DetailedFishSystem` variants have per-species `patternTextureId`; fish materials use preloaded external pattern textures when present and skip disposing shared asset textures during mesh rebuilds. Verified with full `npm run test`, `npm run typecheck`, and `npm run build`.
