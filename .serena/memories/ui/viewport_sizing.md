Aquarium viewport sizing is primarily controlled by CSS in src/styles.css, not by AdvancedAquariumScene. The Three.js scene already reads the container bounds via AdvancedAquariumScene.getViewportSize() and updates on window resize. To make the aquarium fill the viewport, set #canvas-container to width: 100vw and height: 100dvh (with 100vh fallback if desired) and avoid mobile media-query overrides that shrink it back to a square.
