2026-03-17: Reworked fish orientation and locomotion realism in `src/components/DetailedFish.ts` and `src/utils/Boids.ts`. `FishVariant` now carries per-render-path `modelForwardAxis` (`procedural|school|hero`) plus `locomotionProfileId` (`calm-cruiser`, `disk-glider`, `slender-darter`, `goldfish-wobble`). Heading is resolved from the configured forward axis to velocity instead of the old hard-coded `-X` forward. Root motion no longer adds tail-beat yaw; root now handles heading/pitch/bank only, while school fish use small roll/pitch sway and hero fish use a wrapped motion container with optional local tail node animation (`userData.motionNodes`). `BoidsSystem` now supports per-boid tuning via `setBoidTuning`, with runtime params derived from profile-specific cruise speed, cruise bias, boundary arc radius, turn noise, yaw responsiveness, and steering weights. Boundary steering now supports wider return arcs for disk-like fish and faster cruise bias for slender fish while keeping planted-tank sudden turns weak. Added/updated tests in `src/components/DetailedFish.test.ts` and `src/utils/Boids.test.ts`. Verified with `npm run test`, `npm run typecheck`, and `npm run lint`.
